# Antigravity Project Context & Rules

## Workflow Rules (Spec-Kit Cycle)
We follow this Strict Development Cycle for every feature. Do not skip steps.

1. **Specify**: When I ask for a feature, use `specify` tool to define WHAT & WHY.
2. **Clarify**: Ask clarifying questions if anything is ambiguous.
3. **Plan**: Create a technical plan.
4. **Tasks**: Break it down into small checklist items.
5. **Implement**: Write code step-by-step.
6. **Test**: Verify before finishing.
